# Design tokens

The whole system is built from a disciplined palette, two geometric typefaces, and a hard
offset-shadow scale. Surface and ink tokens are CSS variables that flip under a `.dark` class.

## Primary triad

These three colors carry the Bauhaus identity and stay vivid in both light and dark mode.

<Grid>
  <Swatch name="bred" value="#E63329" />
  <Swatch name="bblue" value="#21409A" />
  <Swatch name="byellow" value="#F4C20D" />
  <Swatch name="ink" value="#1C1C1C" />
</Grid>

## Surfaces & ink (theme-aware)

In light mode these are the values below; under `.dark` they invert (paper → near-black,
ink → cream) while keeping the same token names.

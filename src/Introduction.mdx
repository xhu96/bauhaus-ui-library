# Bauhaus UI

A bold, geometric React component library inspired by the **Bauhaus** movement: primary
colors, hard edges, thick black borders, and a relentless grid. 29 accessible components,
built-in light and dark theming, built with TypeScript and Tailwind.

## Install

```bash
npm install bauhaus-ui-library
```

Import the stylesheet once at your app root, then use the components:

```tsx
import 'bauhaus-ui-library/styles.css'
import { Button, Card, GeometricPattern } from 'bauhaus-ui-library'
```

## Theming (light / dark)

The whole library re-themes from a single `.dark` class on a root element. Surfaces
(`paper`, `surface`) and `ink` flip; the red / blue / yellow triad stays vivid.

```js
document.documentElement.classList.toggle('dark')
```

Use the **Theme** toggle in the toolbar above to preview any story in light or dark.

## The component set

- **Foundations**: `Shape`, `ShapeLogo`, `GeometricPattern`, design tokens
- **Core**: `Button`, `Card`
- **Form**: `FormField`, `Input`, `Textarea`, `Select`, `Checkbox`, `Radio`, `Switch`, `Slider`
- **Display**: `Badge`, `Tag`, `Avatar`, `Alert`, `Progress`, `Spinner`, `Divider`, `Tooltip`
- **Navigation**: `Tabs`, `Accordion`, `Navbar`, `Pagination`, `Breadcrumb`
- **Overlay**: `Modal`, `Drawer`, `Toast`

Every component shares a small, predictable API: `color` (`red` / `blue` / `yellow` / `ink`),
`size` (`sm` / `md` / `lg`), and `variant` where it applies.
